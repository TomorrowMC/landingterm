## 已实现功能

### 基础终端功能
1. 命令执行
   - 支持基本的 Shell 命令 (ls, cd, pwd 等)
   - 显示命令输出结果
   - 错误处理和显示

2. 用户界面
   - 类 macOS 终端风格
   - 命令提示符显示当前目录
   - 自动滚动到最新输出
   - 清晰的命令历史显示

3. 目录管理
   - 从用户主目录启动
   - 支持目录导航 (cd 命令)
   - 实时更新当前目录显示
   - 使用 ~ 表示用户主目录

4. 跨平台支持
   - macOS 优先支持 (使用 zsh)
   - 其他平台基础支持 (使用 sh)
